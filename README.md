# PDF-Scraper
A Python script to extract text from PDFs, save it as a text file, and export a list of words and their frequencies to a CSV file for further analysis. Additional functionality will be added soon.

## Dependencies
* PDF Miner (https://github.com/euske/pdfminer)
* Natural Language Toolkit (https://github.com/nltk/nltk)
